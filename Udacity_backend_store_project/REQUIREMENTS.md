## üìò API Endpoints

### Users
| Method | Endpoint | Description |
|--------|-----------|--------------|
| GET | /users | Get all users (Protected) |
| GET | /users/:id | Get a single user by ID (Protected) |
| POST | /users | Create a new user |
| POST | /users/authenticate | Authenticate user and return JWT |
| DELETE | /users/:id | Delete a user (Protected) |

### Products
| Method | Endpoint | Description |
|--------|-----------|--------------|
| GET | /products | Get all products |
| GET | /products/:id | Get product by ID |
| POST | /products | Create a new product (Protected) |
| DELETE | /products/:id | Delete a product (Protected) |

### Orders
| Method | Endpoint | Description |
|--------|-----------|--------------|
| GET | /orders | Get all orders (Protected) |
| GET | /orders/:id | Get order by ID (Protected) |
| POST | /orders | Create a new order (Protected) |
| POST | /orders/:id/products | Add a product to an order (Protected) |
| DELETE | /orders/:id | Delete an order (Protected) |

## üóÉÔ∏è Database Schema

### Users Table
| Column | Type | Constraints |
|--------|------|-------------|
| id | SERIAL | PRIMARY KEY |
| firstname | VARCHAR(100) | NOT NULL |
| lastname | VARCHAR(100) | NOT NULL |
| password_digest | TEXT | NOT NULL |

### Products Table
| Column | Type | Constraints |
|--------|------|-------------|
| id | SERIAL | PRIMARY KEY |
| name | VARCHAR(150) | NOT NULL |
| price | INTEGER | NOT NULL |

### Orders Table
| Column | Type | Constraints |
|--------|------|-------------|
| id | SERIAL | PRIMARY KEY |
| user_id | INTEGER | REFERENCES users(id) |
| status | VARCHAR(50) | CHECK (status IN ('active', 'complete')) |

### Order_Products Table (Many-to-Many)
| Column | Type | Constraints |
|--------|------|-------------|
| id | SERIAL | PRIMARY KEY |
| order_id | INTEGER | REFERENCES orders(id) |
| product_id | INTEGER | REFERENCES products(id) |
| quantity | INTEGER | NOT NULL |

---

## üîê Authentication
All routes that modify or retrieve sensitive data (users, orders) require a **Bearer Token** generated by the `/users/authenticate` endpoint.
